Linux File Hierarchy:
/ (root)   -> above every other directory
    /bin   -> Command Binaries
    /boot  -> Files of Bootloader
    /dev   -> Device Files
    /etc   -> Configurable Files
    /home  -> Non Root Users' (Home) Directory
    /lib   -> Shared Libraries
    /media -> Removable Media
    /mnt   -> Temporarily Mounted Media
    /opt   -> 3rd-Party Applications/Softwares
    /tmp   -> Temporary Files
    /usr   -> User Apps and Data
    /proc  -> Process Information
    /snap  -> Mount & Store Snap Packages... (Snap packages allow users to install, update and manage applications through self contained packages known as snaps)
